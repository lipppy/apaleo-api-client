    Installation\n\nGet started with Apaleo API Client quickly and easily. The library supports Python 3.10+ and can be installed using your preferred package manager.\n\n## \ud83d\udce6 Package Installation\n\n### Using pip (recommended)\n\nbash\npip install apaleo-api-client\n\n\n### Using Poetry\n\nbash\npoetry add apaleo-api-client\n\n\n### Using pipenv\n\nbash\npipenv install apaleo-api-client\n\n\n### Using conda\n\nbash\nconda install -c conda-forge apaleo-api-client\n\n\n## \ud83d\udd27 Development Installation\n\nFor development or contributing to the project:\n\n### Clone the Repository\n\nbash\ngit clone https://github.com/apaleo/apaleo-api-client-python.git\ncd apaleo-api-client-python\n\n\n### Install with Poetry (Recommended)\n\nbash\n# Install with all development dependencies\npoetry install --with dev,test,lint,docs\n\n# Activate the virtual environment\npoetry shell\n\n\n### Install with pip\n\nbash\n# Create virtual environment\npython -m venv venv\nsource venv/bin/activate # On Windows: venv\\Scripts\\activate\n\n# Install in development mode\npip install -e \".[dev,test,lint,docs]\"\n\n\n## \ud83d\udd0d System Requirements\n\n### Python Version\n- Python 3.10 or higher\n- Python 3.13 is fully supported\n- Tested on CPython (PyPy support coming soon)\n\n### Operating Systems\n- \u2714\ufe0f Linux (Ubuntu, CentOS, Alpine)\n- \u2714\ufe0f macOS (Intel and Apple Silicon)\n- \u2714\ufe0f Windows (10, 11, Server)\n\n### Dependencies\n\nCore dependencies (automatically installed):\n\n\nhttpx>=0.28.1 # Async HTTP client\npydantic>=2.12.5 # Data validation and serialization\ndacite>=1.9.2 # Dataclass conversion\nhttpx-retries>=0.4.6 # HTTP retry middleware\npolyfactory>=3.3.0 # Testing factories\n\n\n## \ud83d\uddc2\ufe0f Verify Installation\n\nTest that the installation was successful:\n\npython\nfrom apaleoapi import ApaleoAPIClient\nfrom apaleoapi import __version__\n\nprint(f\"Apaleo API Client version: {__version__}\")\n\n# Test client initialization (dry-run mode)\nclient = ApaleoAPIClient(\n client_id=\"test\",\n client_secret=\"test\",\n dry_run=True\n)\n\nprint(\"Installation successful!\")\n\n\n## \ud83c\udfe1 Getting API Credentials\n\nBefore using the client, you'll need Apaleo API credentials:\n\n### 1. Register with Apaleo\n1. Visit the Apaleo Developer Portal\n2. Create a developer account or sign in\n3. Navigate to \"Applications\" or \"API Keys\"\n\n### 2. Create an Application\n1. Click \"Create Application\"\n2. Enter your application details:\n - Name: Your application name\n - Description: Brief description of your integration\n - Redirect URI: For OAuth flows (if needed)\n\n### 3. Get Your Credentials\nAfter creating the application, you'll receive:\n- Client ID: Public identifier for your application\n- Client Secret: Private secret (keep secure!)\n\n### 4. Configure Scopes\nSelect the appropriate API scopes for your use case:\n- core:read - Read access to Core API\n- core:write - Write access to Core API \n- identity:read - Read access to Identity API\n- Additional scopes as needed\n\n## \u2699\ufe0f Configuration\n\n### Environment Variables\n\nStore your credentials securely using environment variables:\n\nbash\n# .env file\nAPALEO_CLIENT_ID=your-client-id-here\nAPALEO_CLIENT_SECRET=your-client-secret-here\nAPALEO_TIMEOUT=30\nAPALEO_MAX_RETRIES=3\nAPALEO_MAX_CONCURRENT=5\n\n\nLoad them in your application:\n\npython\nimport os\nfrom apaleoapi import ApaleoAPIClient\n\nclient = ApaleoAPIClient(\n client_id=os.getenv(\"APALEO_CLIENT_ID\"),\n client_secret=os.getenv(\"APALEO_CLIENT_SECRET\"),\n timeout=float(os.getenv(\"APALEO_TIMEOUT\", 30)),\n retries=int(os.getenv(\"APALEO_MAX_RETRIES\", 3)),\n max_concurrent=int(os.getenv(\"APALEO_MAX_CONCURRENT\", 5))\n)\n\n\n### Configuration File\n\nAlternatively, use a configuration file:\n\nyaml\n# apaleo_config.yaml\napaleo:\n client_id: \"your-client-id\"\n client_secret: \"your-client-secret\"\n timeout: 30.0\n retries: 3\n max_concurrent: 5\n dry_run: false\n \nlogging:\n level: \"INFO\"\n format: \"%(asctime)s - %(name)s - %(levelname)s - %(message)s\"\n\n\npython\nimport yaml\nfrom apaleoapi import ApaleoAPIClient\n\nwith open(\"apaleo_config.yaml\") as f:\n config = yaml.safe_load(f)\n\nclient = ApaleoAPIClient(**config[\"apaleo\"])\n\n\n## \ud83d\udee1\ufe0f Security Best Practices\n\n### Credential Management\n- \u26a0\ufe0f Never commit credentials to version control\n- Use environment variables or secure secret management\n- Rotate credentials regularly\n- Use different credentials for development/staging/production\n\n### Network Security\n- Always use HTTPS (enforced by default)\n- Consider using a VPN for sensitive environments\n- Monitor API access logs\n\n### Application Security\n- Validate all user inputs before API calls\n- Implement proper error handling\n- Log security events appropriately\n- Follow the principle of least privilege for API scopes\n\n## \ud83d\udd25 Quick Start Example\n\nPut it all together with a complete example:\n\npython\nimport asyncio\nimport os\nfrom apaleoapi import ApaleoAPIClient\nfrom apaleoapi.apaleo.core.v1.dataclasses.inventory import PropertyListParams\n\nasync def main():\n # Initialize client with environment variables\n client = ApaleoAPIClient(\n client_id=os.getenv(\"APALEO_CLIENT_ID\"),\n client_secret=os.getenv(\"APALEO_CLIENT_SECRET\")\n )\n \n try:\n # Fetch first 10 properties\n params = PropertyListParams(page_size=10)\n properties = await client.core.v1.inventory.list_properties(params)\n \n print(f\"Found {properties.count} total properties\")\n for prop in properties.items:\n print(f\"- {prop.name} ({prop.code})\")\n \n finally:\n # Always clean up\n await client.aclose()\n\nif __name__ == \"__main__\":\n asyncio.run(main())\n\n\n## \ud83d\udd04 Upgrading\n\n### Check Current Version\n\npython\nfrom apaleoapi import __version__\nprint(__version__)\n\n\n### Upgrade to Latest\n\nbash\npip install --upgrade apaleo-api-client\n\n\n### Upgrade with Poetry\n\nbash\npoetry update apaleo-api-client\n\n\n## \ud83c\udf86 Next Steps\n\nNow that you have the library installed:\n\n1.
