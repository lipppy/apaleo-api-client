# API Reference

This section provides detailed documentation for all API endpoints supported by the Apaleo API Client. Each endpoint includes:

- **Endpoint URL** and HTTP method
- **Request parameters** with types and descriptions
- **Response schema** with example data
- **Error handling** and possible exceptions

## Core API

This section covers the Core API endpoints, which provide access to Apaleo's core functionalities such as properties, inventory, rates, and units management. The Core API is the foundation of the Apaleo ecosystem and is essential for most integrations.
