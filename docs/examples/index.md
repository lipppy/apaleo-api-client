# Overview

Use these examples to choose the OAuth flow that fits your app.

- [Client Credentials Flow](client_credentials_flow.md): Best for server-to-server integrations.
- [Authorization Code Flow](authorization_code_flow.md): Best when accessing data on behalf of a user.

- [OAuth 2.0 - Authorization code grant flow](https://apaleo.dev/guides/oauth-connection/auth-code-grant.html)
